🛵 Swiggy Sales Dashboard
An interactive sales analytics dashboard built with Power BI to analyze Swiggy's food delivery performance across India.
Show Image

📊 Overview
This dashboard provides a comprehensive view of Swiggy's sales data, helping stakeholders track performance metrics, identify trends, and make data-driven decisions across cities and states.

✨ Key Metrics
MetricValue💰 Total Sales₹ 53.01M⭐ Average Rating4.34🛒 Avg Order Value₹ 268.51👍 Rating Count5.59M📦 Total Orders197,430

📈 Dashboard Features

Monthly Sales Trend – Track revenue performance from Jan to Aug
Daily Sales Trend – Identify peak order days across the week
Weekly Sales Trend – Monitor week-by-week growth over 36 weeks
Top 5 Cities by Sales – Bengaluru, Lucknow, Hyderabad, Mumbai, New Delhi
Sales by Food Type – Non-Veg (64%) vs Veg (36%) breakdown
Sales by State – Geographic heatmap across India
Quarterly Performance Table – Sales, Rating & Orders by Q1, Q2, Q3
