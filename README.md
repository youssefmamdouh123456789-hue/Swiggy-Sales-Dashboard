# Swiggy Sales Dashboard

This is an Excel-based interactive dashboard used to analyze sales performance, ratings, and order trends for Swiggy data.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers

## Project Preview

The dashboard screenshot is included in this repository.

## Key Insights

* Sales analysis across cities and states
* Customer ratings and order trends
* Food category performance
